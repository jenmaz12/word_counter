PROJECT TITLE: WordCounter
PURPOSE OF PROJECT: Count the number of words in a file.
VERSION or DATE: 6/28/20
HOW TO START THIS PROJECT: Compile WordCounter.java and LineReader.java and run main in WordCounter.
AUTHORS: Jennifer Mazanec
USER INSTRUCTIONS: Enter the file name if in the same directory or the file path if in a different directory in the input dialog. Click OK. The program will show a message dialog box stating the number of words in the file.
